# Contributing

## Setup

```bash
git clone https://github.com/kairav7220/BlogWriter.git
cd BlogWriter
pip install -r requirements.txt
```

## Development

- Fork the repo, create a feature branch.
- Test with `streamlit run agent.py`
- Ensure `.env` is in `.gitignore` before committing.

## PR Guidelines

- One feature/fix per PR.
- Keep it single-file if possible — that's the appeal.
